# LinkedIn Optimization
*Profile setup for Cloud / DevOps roles*

**Source**: [YouTube – LinkedIn para devs](https://www.youtube.com/watch?v=8bYDombV5R0&t=1627s)

---

## Profile Basics
*Account-level setup*

**Verified profile** – Activate identity verification  
**Custom URL** – `linkedin.com/in/nombre-apellido` (no random numbers)  
**Contact info** – Add email and GitHub link  
**Activity** – Post and interact regularly (LinkedIn is a social network)  
**Network** – Connect heavily with people in your target area (e.g., Cloud)

---

## Headline
*Line below the name*

Order: **target role → keywords → tools → specialty**

Example: `AWS Engineer | Cloud · DevOps · Backend | AWS, Terraform, Docker, Python`

---

## About
*Short intro paragraph*

Briefly state:
- Current focus (entering DevOps / Cloud world)
- Certifications you hold
- What you're looking for

Keep it short — no walls of text.

---

## Experience
*Per-role entries*

**Hyperlink** – Tag the company so its logo appears  
**Filter** – Keep only roles relevant to the target position  
**Format** – Bullet list (~6 points), no paragraphs  
**Content per bullet** – Function + how you did it + achieved outcome  
**Outcomes** – Quantify when possible (faster system, more scalable, X% improvement)

---

## Skills
*Skills section*

Add **all** soft skills (communication, teamwork…) and **all** tools (AWS, GitHub, Docker…).  
More = better → passes LinkedIn's keyword bot filter.

---

## Certifications
*Only the representative ones*

Don't dump every course. Keep only the certifications that signal the target role.

---

## Languages
*Always include*

List every language you speak with proficiency level.

---

## Visuals
*Banner and photo*

**Banner** – Name + target role + email  
**Photo** – Plain light background, head + shoulders visible, neutral expression

## Job Search Trick
*Filter by post age via URL*

LinkedIn's job search uses `f_TPR=rN` where `N` = seconds since posted.

| Window | Value |
|---|---|
| Last 24 hours | `r86400` |
| Last hour | `r3600` |
| Last 10 min | `r600` |

Smaller window = fewer applicants = better odds.

```
https://www.linkedin.com/jobs/search/?keywords=designer&f_TPR=r3600
```

Replace `86400` with `3600` (or any number of seconds) in the URL bar.
