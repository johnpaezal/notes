# AWS S3
*Object storage service*

## What is S3
*Infinitely scalable object storage*

**S3 (Simple Storage Service)** – Store and retrieve any amount of data  
**Bucket** – Container for objects; name is globally unique  
**Object** – File + metadata stored in a bucket  
**Key** – Object's unique path within the bucket (`images/user/avatar.png`)  
**Prefix** – Folder-like path segment (not a real folder; just a key prefix)  

**Durability**: 99.999999999% (11 nines) — objects replicated across AZs.

---

## Storage Classes
*Cost vs access frequency trade-off*

| Class | Use Case | Retrieval |
|-------|----------|-----------|
| Standard | Frequently accessed | Instant |
| Standard-IA | Infrequent access | Instant (fee/GB) |
| Glacier Instant | Archive, rare access | Instant |
| Glacier Flexible | Archive, hours OK | 1–12h |
| Glacier Deep Archive | Long-term, years | 12–48h |

## Basic Operations
*Upload, download, list*

```bash
# AWS CLI
aws s3 mb s3://my-bucket                    # create bucket
aws s3 ls s3://my-bucket                    # list objects
aws s3 cp file.txt s3://my-bucket/          # upload
aws s3 cp s3://my-bucket/file.txt .         # download
aws s3 sync ./local-dir s3://my-bucket/     # sync directory
aws s3 rm s3://my-bucket/file.txt           # delete
```

```python
# Boto3
import boto3

s3 = boto3.client("s3")

s3.upload_file("local.txt", "my-bucket", "remote/path.txt")
s3.download_file("my-bucket", "remote/path.txt", "local.txt")

# Generate pre-signed URL (temporary public access)
url = s3.generate_presigned_url(
    "get_object",
    Params={"Bucket": "my-bucket", "Key": "report.pdf"},
    ExpiresIn=3600    # 1 hour
)
```

## Bucket Policies and Access
*Control who can access objects*

```json
{
  "Version": "2012-10-17",
  "Statement": [{
    "Effect": "Allow",
    "Principal": "*",
    "Action": "s3:GetObject",
    "Resource": "arn:aws:s3:::my-public-bucket/*"
  }]
}
```

**Block Public Access** – Bucket-level setting; enabled by default; disables all public access.  
**ACL** – Legacy per-object access control (avoid; use bucket policies instead).  
**Pre-signed URL** – Time-limited URL for private object access without making bucket public.

## Common Patterns
*Practical use cases*

**Static Website Hosting** – Enable on bucket → set index.html → point Route 53 or CloudFront  
**Event Notifications** – Trigger Lambda on `s3:ObjectCreated:*`  
**Versioning** – Keep all versions of an object; restore deleted files  
**Lifecycle Rules** – Auto-transition to cheaper class after N days; delete after expiry  

```bash
# Enable versioning
aws s3api put-bucket-versioning \
  --bucket my-bucket \
  --versioning-configuration Status=Enabled
```
